# AI Agent Usage

## Why I Used AI Agent

복잡한 컴퓨터비전 분석 파이프라인과 FastAPI/React 웹서비스 흐름을 빠르게 구조화하기 위해 AI Agent를 활용했습니다.

## AI Agent Assisted

- 요구사항 분해
- FastAPI API 구조 초안 작성
- SSE progress streaming 구조 설계 보조
- YOLO/OpenCV/SciPy 분석 파이프라인 아이디어 정리
- React UI 구성 보조
- 오류 원인 후보 정리
- README 문서화 보조

## My Decisions and Verification

- 단일 카메라 기반 분석의 한계를 고려해 결과를 “정밀 측정”이 아니라 “추정 및 시각화”로 포지셔닝했습니다.
- 골대 규격 기반 scale estimation 방식을 적용했습니다.
- 업로드, 진행률 표시, 결과 이미지 생성 흐름을 직접 실행해 확인했습니다.
- 제출용 문서에서 과장 표현을 제거했습니다.
